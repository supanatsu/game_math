# 🎮 High-Low Battle Number

เกมทายตัวเลขแบบแข่งขัน 2 คน (2-Player Local Multiplayer) ที่ผู้เล่นจะต้องแข่งกันประมวลผลและหา "ตัวเลขปริศนา" ให้เจอก่อนที่เวลาจะหมดลง โดยระบบจะมีคำใบ้บอกว่าตัวเลขที่ทายไปนั้น **"มากกว่า"** หรือ **"น้อยกว่า"** คำตอบที่ถูกต้อง 

ตัวเกมถูกพัฒนาด้วย HTML, CSS และ JavaScript โดยไม่ต้องพึ่งพาไลบรารีภายนอก สามารถเปิดเล่นผ่านเว็บเบราว์เซอร์ได้ทันที

---

## ✨ ฟีเจอร์หลัก (Features)

* **⚔️ 2-Player Split-Screen:** รองรับการเล่น 2 คนในหน้าจอเดียว แข่งขันกันแบบเรียลไทม์
* **⏱️ Countdown Timer:** ระบบจับเวลาถอยหลัง (ค่าเริ่มต้น 60 วินาที) เพิ่มความกดดันและตื่นเต้น
* **📜 Guess History (Log):** มีระบบบันทึกประวัติการทายของแต่ละคน พร้อมสัญลักษณ์บอกใบ้:
  * `⬆️` หมายถึง คำตอบที่ถูกต้อง **มากกว่า** เลขที่ทายมา
  * `⬇️` หมายถึง คำตอบที่ถูกต้อง **น้อยกว่า** เลขที่ทายมา
* **♾️ Unlimited Guesses:** ผู้เล่นสามารถทายกี่ครั้งก็ได้จนกว่าจะเจอคำตอบหรือหมดเวลา
* **⌨️ Keyboard Support:** รองรับการกดปุ่ม `Enter` เพื่อส่งคำตอบ ทำให้เกมเพลย์ลื่นไหลและรวดเร็ว

---

## 🚀 วิธีการติดตั้งและรันเกม (How to Run)

เกมนี้เป็นไฟล์ Static Web Application ไม่จำเป็นต้องติดตั้งเซิร์ฟเวอร์หรือ Node.js

1. คัดลอกโค้ดเกมไปสร้างเป็นไฟล์ชื่อ `index.html`
2. ดับเบิลคลิกที่ไฟล์ `index.html` เพื่อเปิดผ่านเว็บเบราว์เซอร์ที่คุณใช้งาน (เช่น Google Chrome, Microsoft Edge, Safari หรือ Firefox)
3. เริ่มเล่นได้ทันที!

---

## 🕹️ วิธีการเล่น (How to Play)

1. เมื่อเปิดเกม ระบบจะสุ่มตัวเลข 1 ชุด (ค่าเริ่มต้นคือ 1 ถึง 1000)
2. **Player 1** (ฝั่งซ้าย) และ **Player 2** (ฝั่งขวา) พิมพ์ตัวเลขลงในช่องของตัวเอง
3. กดปุ่ม **"ส่ง!"** หรือกด **`Enter`**
4. สังเกตผลลัพธ์ในกล่องประวัติการทาย:
   * ถ้าระบบบอกว่า `⬆️ 500 -> มากกว่านี้` ให้ทายเลขที่มากกว่า 500
   * ถ้าระบบบอกว่า `⬇️ 800 -> น้อยกว่านี้` ให้ทายเลขที่น้อยกว่า 800
5. ใครที่หาตัวเลขที่ถูกต้องเจอเป็นคนแรกก่อนเวลาหมด จะเป็น **ผู้ชนะ!**
6. หากต้องการเล่นรอบใหม่ ให้กดปุ่ม **"🔄 รีเซ็ตเกมใหม่"** ด้านบน
